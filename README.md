<h1>Developer Tool for Valtio applications</h1>
deValtio is a Google Chrome extension that serves as a visualization and debugging tool for <a href="https://github.com/pmndrs/valtio">Valtio, a proxy-based state management library</a> for <a href="https://reactjs.org/">Reactjs</a>.

<h2>Getting Started/Installation</h2>
<p>1. Please install the chrome extension - #link here</p>
<p>2. Open your application on the Chrome Browser and start debugging with the devtool. 
    <ul>
        <li>Only support React Applications</li>
        <li>Only intended for projects in development mode?</li>
    </ul>
</p>

<h2>Contributions Welcome</h2>
DeValtio is an open-source project and welcomes contributions. We encourage you to fork the repository, create a feature branch and submit a pull request. 

Also, please note our development tool is still in beta. If you find any issues, please report them in the issues tab or submit a pull request. We welcome all advice, improvements and fixes.
